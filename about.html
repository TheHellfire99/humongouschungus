Vocabulary:

  • HTML: Hyper-Text markup language.
  • CSS: Cascading Style Sheet
  • Attributes: Attributes are extra information inside of an HTML element that provides additional data about the element.
  • Index: A special HTML filename that almost every web server recognizes as the "home page" of a website.
  • Class: Gives your tag a type. You can use it to specifically target and style a tag!
